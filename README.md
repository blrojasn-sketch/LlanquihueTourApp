LlanquihueTourApp

Descripción

Sistema desarrollado en Java para gestionar servicios turísticos y entidades 
asociadas a la agencia Llanquihue Tour.

La aplicación implementa:
- Interfaces
- Herencia
- Polimorfismo
- ArrayList
- instanceof
- GUI con JOptionPane

Clases e Interfaces

Interfaz

- ExcursionCultural

### Entidades
- GuiaTuristico
- Vehiculo
- ColaboradorExterno

Gestión
- GestorEntidades

Interfaz Gráfica
- Main

Ejecución

1. Abrir el proyecto en IntelliJ IDEA.
2. Verificar que el JDK esté configurado.
3. Abrir la clase `Main.java`.
4. Ejecutar el método `main()`.
5. Utilizar las ventanas emergentes para registrar entidades y visualizar 
los resultados.
